# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.4] - 2026-01-11

### Changed
- Update @fistware/logger to v2.0.5 for improved logging capabilities

## [1.0.3] - 2025-12-12

### Changed
- Fix import paths in `useCalendarNavigation` hook.

## [1.0.2] - 2025-12-12

### Changed
- Add `exports` option to package.json to export the package.
- Add `types` option to package.json to export the types.
- Add `default` option to package.json to export the default.

## [1.0.1] - 2025-12-12

### Changed
- Add `declaration` option to tsconfig.json to generate declaration files.

## [1.0.0] - 2025-12-12

### Added

- Initial release of the React components package.
- Add `useCalendar` hook to handle calendar functionality.
- Add `useCalendarNavigation` hook to handle calendar navigation.
- Add `useFloatingCalendarInput` hook to handle floating calendar input.
- Add `useDashboardCalendar` hook to handle dashboard calendar.
- Add `useAlert` hook to handle alert functionality.
- Add `useCopyToClipboard` hook to handle copy to clipboard functionality.
- Add `useLoading` hook to handle loading functionality.
- Add `useSwipe` hook to handle swipe functionality.
- Add `LoadingContext` context to handle loading state.
- Add `useClickOutside` hook to handle click outside functionality.
- Add `ErrorIcon` component to handle error icon functionality.
- Add `WarningIcon` component to handle warning icon functionality.
- Add `SuccessIcon` component to handle success icon functionality.
- Add `LeftArrow` component to handle left arrow functionality.
- Add `RightArrow` component to handle right arrow functionality.
- Add `XIcon` component to handle X icon functionality.
